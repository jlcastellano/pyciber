5.1 Plataformas CTF y Práctica

Los CTF (Capture The Flag) son competiciones de ciberseguridad donde se resuelven retos para obtener "banderas" (flags). Son la mejor forma de practicar habilidades de hacking ético.

Plataformas para principiantes

PicoCTF
URL: https://picoctf.org
Nivel: Principiante
Características: Ejercicios guiados, múltiples categorías (crypto, forensics, web, binary), ideal para empezar
OverTheWire
URL: https://overthewire.org
Nivel: Principiante a intermedio
Wargames destacados:
  • Bandit: Comandos básicos de Linux
  • Natas: Vulnerabilidades web
  • Krypton: Criptografía
TryHackMe
URL: https://tryhackme.com
Nivel: Todos los niveles
Características: Rooms guiadas paso a paso, paths de aprendizaje estructurados, máquinas vulnerables en la nube

Plataformas intermedias y avanzadas

HackTheBox
URL: https://hackthebox.eu
Nivel: Intermedio a avanzado
Características: Máquinas realistas, retos de diferentes categorías, comunidad muy activa, Pro Labs para entornos empresariales
VulnHub
URL: https://vulnhub.com
Nivel: Intermedio
Características: Máquinas virtuales descargables, Boot2Root challenges, práctica offline
Root-Me
URL: https://root-me.org
Nivel: Todos los niveles
Características: Cientos de retos, múltiples categorías, documentación y soluciones disponibles
